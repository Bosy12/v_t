Description 
1. project E2EFunction target is to automate basic google search operation for a key word and validate that the ads links has a title 
2. project v_t_main target is to automate remote api consuming for (get user , get posts and place and new post )


prerequists 
1. eclipse 
2. install the following eclipse : Cucumper , TestNG and Junit 

To to run 
1. create worksapce 
2. import the project 
3. to run E2EFunction > right click on the feature file 'Features/automated' and run as cucmber feature  
4. to run v_t_main > right click on the testng.xml and run as TestNG
