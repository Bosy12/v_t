*Description*

1. project E2EFunction target is to automate basic google search operation for a keyword and validate that the ads links has a title

2. project v_t_main target is to automate remote api consuming for (get user , get posts and place and new post )
  ![image](https://user-images.githubusercontent.com/46898914/181996039-25cdd469-034b-49c2-a739-8d1df8e908a4.png)

*Prerequisite*

1. eclipse
2. install the following eclipse : Cucumber , TestNG and Junit ( from help menu > market place search for each of them and install )

*To to run*

1. create workspace
2. import the project
3. to run E2EFunction > right click on the feature file 'Features/automated' and run as cucumber  feature
4. to run v_t_main > right click on the testng.xml and run as TestNG
